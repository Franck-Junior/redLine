English :
Thank you for choosing Upssearchtech !

This minimal readme will only explain you how to install/uninstall Upssearchtech.

Launch the script like this :
./install.sh "/path/you/want"

Exit codes :

0 : Your install went fine.
1 : There's the wrong amount of parameters (should be one).
2 : You don't have the sufficient rights to install Upssearchtech.

We hope that you will have fun with Upssearchtech, but if in any case you wanted to
part ways with us, just launch uninstall.sh and follow the instructions.

French :

Merci d'avoir choisi Upssearchtech !

Ce readme vous expliquera seulement comment installer/désinstaller Upssearchtech.

Lancez le script comme ceci :
./install.sh "/chemin/choisi"

Codes d'erreur :

0 : Tout s'est bien passé.
1 : Il n'y a pas le bon nombre de paramètres (il devrait y en avoir un).
2 : Vous n'avez pas les droits suffisants pour installer Upssearchtech.

On espère que vous vous amuserez bien avec Upssearchtech, mais si d'aventure vous
vouliez nous abandonner, lancez uninstall.sh et suivez les instructions.
